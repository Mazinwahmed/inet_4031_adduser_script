# INET 4031 Add User Script

This repository contains a shell script designed to automate the process of adding users to an Ubuntu Server environment. It ensures consistent user creation, group assignment, and home directory setup.

## Description

The `adduser_script.sh` simplifies administrative tasks by handling user creation via the command line. This project was developed as part of the INET 4031 course requirements.

## Prerequisites

* **Operating System:** Ubuntu Server VM
* **Permissions:** You must have `sudo` or root privileges to execute this script.
* **Git:** Ensure Git is installed to manage this repository.
